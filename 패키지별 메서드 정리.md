## NumPy: 수치 연산 및 행렬 처리
고성능 수치 계산을 위한 라이브러리. 

<br>

* **`np.array()`**: 리스트나 튜플 객체를 N차원 배열(`ndarray`)로 변환하여 효율적인 행렬 연산을 함.
* **`np.dot()` / `.dot()`**: 두 배열의 행렬 곱을 수행하며, 선형 회귀 가설 함수 $H(X) = WX + b$ (직·방) 계산.
* **`.T`**: 행렬의 전치를 수행하여 경사 하강법 수식 계산 시 행렬 간의 차원을 맞추기 위해 사용.
* **`np.random.seed()`**: 난수 생성기의 시드값을 고정하여 코드를 실행할 때마다 동일한 난수가 생성되도록 결과의 재현성을 보장.

<br>

## Pandas: 데이터 탐색 및 전처리
표 형태의 데이터를 처리하고 분석하는 라이브러리

<br>

* **`pd.DataFrame()`**: 딕셔너리나 NumPy 배열 데이터를 행과 열이 있는 표 형태의 구조로 변환. 
* **`.head(n)`**: 데이터프레임의 상위 `n`개 행을 출력하여 데이터의 구성을 빠르게 파악. 
```python
diabetes = load_diabetes()
df = pd.DataFrame(diabetes.data, columns=diabetes.feature_names)
#'PROGRESS' 열에 목표값(당뇨병) 데이터 추가
df['PROGRESS'] = diabetes.target

print("--- 당뇨병 데이터 집합 샘플 ---")
print(df.head())
```

<br>

* **`.info()`**: 전체 행 개수, 컬럼별 데이터 타입, 결측치 여부 등 데이터프레임의 요약 정보를 확인.
```python
df.info()
``` 

<br>

* **`.drop(labels, axis)`**: 특성($X$)과 라벨($y$)을 분리하거나 불필요한 행/열을 삭제할 때 사용.
```python
# 특성(X)과 목표값(y)을 분리합니다.
X = df.drop('PROGRESS', axis=1)
y = df['PROGRESS']
```

## Scikit-learn: 머신러닝 워크플로우
머신러닝 모델 구현 및 평가를 위한 라이브러리

<br>

* **`train_test_split()`**: 전체 데이터를 학습용(Train)과 평가용(Test)으로 분리하여 모델의 유효성을 객관적으로 검증.
> 보통 7:3 8:2로 나누며, 예를들어 100개의데이터를 가지고 있다면 80개로 학습시키고 나머지 20개로 테스트를하여 정확도를 보는것임.
* **`LinearRegression()`**: 선형 회귀 알고리즘을 구현한 클래스로, 학습(`.fit()`)과 예측(`.predict()`) 기능을 제공. 
* **`mean_squared_error()`**: 예측값과 실제값의 차이인 오차 제곱의 평균(MSE)을 계산하여 모델의 성능을 수치화함. 

<br>

## Matplotlib: 데이터 시각화
데이터 분석 결과를 직관적인 그래프로 표현하는 라이브러리

<br>

* **`plt.figure()`**: 그래프의 크기를 설정하여 시각화 결과의 가독성을 조절함
```python
plt.figure(figsize=(10, 10))
```

<br>

* **`plt.plot()`**: 직선이나 곡선을 그려 모델의 이상적인 기준선 등을 표시함
* **`plt.title() / xlabel() / ylabel()`**: 그래프에 제목과 축 설명을 추가하여 정보 전달력을 높임
```python
plt.title("GT vs. Preds (Diabetes Progression)")
plt.xlabel("Model Prediction")
plt.ylabel("Ground Truth")
```

<br>

## Seaborn: 데이터 시각화
데이터 분석 결과를 직관적인 그래프로 표현하는 라이브러리

<br>

* **`sns.scatterplot()`**: 데이터를 점으로 표현하여 예측값과 실제값의 상관관계를 시각화함.
* **`sns.regplot()`**: 산점도와 함께 데이터의 경향성을 나타내는 회귀선을 그려 모델의 성능을 직관적으로 보여줌.
